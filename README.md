# Analisis-Sentimen-Aplikasi-Shoope-di-Play-Store
Penelitian ini menganalisis sentimen ulasan pengguna aplikasi Shopee di Google Play Store menggunakan teknik Natural Language Processing (NLP). Data ulasan dikumpulkan melalui scraping, lalu diklasifikasikan ke dalam tiga kategori sentimen: positif, negatif, dan netral.
