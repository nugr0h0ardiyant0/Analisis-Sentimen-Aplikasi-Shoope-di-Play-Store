# Analisis-Sentimen-Aplikasi-Shoope-di-Play-Store
Penelitian ini menganalisis sentimen ulasan pengguna aplikasi Shopee di Google Play Store menggunakan tiga pendekatan pemodelan: SVM + TF-IDF, Random Forest + Word2Vec, dan fine-tuning IndoBERT. Data ulasan dilabeli secara otomatis berdasarkan skor rating (≤2 = negative, 3 = neutral, ≥4 = positive), kemudian divalidasi silang menggunakan model pretrained mdhugol/indonesia-bert-sentiment-classification untuk mendeteksi konflik label. Tahap preprocessing mencakup pembersihan teks, normalisasi slang menggunakan kamus alay (colloquial-indonesian-lexicon), dan penghapusan stopwords Bahasa Indonesia. Hasil ketiga model dibandingkan berdasarkan akurasi dan classification report untuk menentukan pendekatan terbaik dalam mengklasifikasikan sentimen ulasan berbahasa Indonesia.
